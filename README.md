## Run Backend

1. Go to `/backend` folder in Terminal
2. Run `go mod tidy`
3. Run `go run main.go`
4. Backend API will be run on `http://localhost:8000`

## Run Frontend

1. Go to `/frontend` folder in Terminal
2. Run `npm install`
3. Run `npm start`
4. Page will be run on `http://localhost:3000`

## API Spec

API Spec is available in `BookCabin.postman_collection.json` file. Open it in Postman.
